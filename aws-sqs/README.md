# nodejs-aws-sqs
A simples Node.js project integrated with AWS SQS (Simple Queue Service)

## How to Run

1. git clone
2. cd into folder
3. npm install
4. copy .env.example as .env
5. fill .env variables
6. run producer.js
7. run consumer.js
